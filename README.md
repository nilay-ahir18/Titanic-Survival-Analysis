
# 🚢 Titanic Survival Analysis

## 📌 Project Overview

This project focuses on analyzing the Titanic dataset to understand the factors that influenced passenger survival. Using data analysis and visualization techniques, we identify patterns based on features such as gender, age, and passenger class.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Conduct Exploratory Data Analysis (EDA)
* Visualize survival patterns
* Identify key factors affecting survival

---

## 📂 Dataset

* Source: Kaggle Titanic Dataset
* File Used: `train.csv`

The dataset includes information such as:

* PassengerId
* Survived
* Pclass
* Name
* Sex
* Age
* Fare
* Embarked

---

## 🛠️ Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## ⚙️ Steps Performed

### 1. Data Loading

* Imported dataset using Pandas

### 2. Data Cleaning

* Handled missing values in Age and Embarked
* Converted categorical data (Sex) into numerical format

### 3. Exploratory Data Analysis (EDA)

* Analyzed survival distribution
* Compared survival by gender and passenger class
* Studied age distribution

### 4. Data Visualization

* Countplot for survival
* Bar charts for gender and class
* Histogram for age
* Heatmap for correlation

---

## 📊 Key Insights

* Female passengers had a higher survival rate than males
* Passengers in higher classes (1st class) had better survival chances
* Younger passengers had slightly higher survival probability
* Fare and class were positively correlated with survival

---

## 📈 Sample Visualizations

* Survival Count Plot
* Survival by Gender
* Survival by Passenger Class
* Age Distribution
* Correlation Heatmap

---

## 🧠 Conclusion

The analysis shows that gender and passenger class were the most important factors influencing survival on the Titanic. Females and first-class passengers had significantly higher survival rates compared to others.
